# Хоолой Клон — дуугаар ярьдаг туслах апп

15–30 секунд хүний хоолой бичиж аваад **тэр хоолойг хуулбарлаж**, асуусан
асуултад **тэр хүний хоолойгоор, монголоор дуугаар хариулдаг** Android апп.

## Хэрхэн ажилладаг вэ

1. **Хоолой бүртгэх** — 15–30 сек бичлэг хийж, ElevenLabs дээр хоолойг
   хуулбарлана (Instant Voice Cloning).
2. **Асуух** — микрофон дээр дараад асуултаа хэлнэ → OpenAI Whisper
   яриаг текст болгоно.
3. **Хариулах** — OpenAI (GPT) монголоор хариулт боловсруулна.
4. **Дуугаар** — ElevenLabs хуулбарласан хоолойгоор хариултыг уншиж тоглуулна.

> Бүх API түлхүүр зөвхөн таны утсан дээр хадгалагдана, өөр хаашаа ч
> илгээгддэггүй.

## Юу шаардлагатай вэ (API түлхүүр)

Апп ажиллахын тулд 2 түлхүүр хэрэгтэй. Эдгээрийг апп дотроо "Тохиргоо"
хэсэгт нэг удаа оруулна:

- **OpenAI** — https://platform.openai.com → API keys (`sk-...`).
  Яриа таних + хариулт боловсруулахад.
- **ElevenLabs** — https://elevenlabs.io → Profile → API key.
  Хоолой хуулбарлах + дуу гаргахад. *Хоолой хуулбарлах (cloning) функц нь
  төлбөртэй багц (Starter ба түүнээс дээш) шаарддагийг анхаарна уу.*

Хоёулаа хэрэглэснээр бага хэмжээний төлбөр гарна (ихэвчлэн нэг асуултад
центийн хэдэн арав).

## .apk файл хэрхэн гаргах вэ

Энэ бол бэлэн эх код. Үүнээс .apk гаргах **2 зам** бий.

### Зам A — GitHub Actions (компьютерт юм суулгахгүй, хамгийн хялбар)

1. github.com дээр шинэ (хувийн) repository үүсгэ.
2. Энэ хавтасны бүх файлыг тэр repo руу upload хий (эсвэл push хий).
3. Repo-гийн **Actions** таб руу ор → "Build APK" → **Run workflow** дар.
4. 5–10 минутын дараа ажил дуусна. Тухайн ажлыг дараад доороос
   **Artifacts → hooloi-clone-apk** татаж аваарай. Дотор нь
   `app-release.apk` байгаа.
5. Тэр .apk-г Android утсандаа хуулж суулга (Тохиргооноос "тодорхойгүй
   эх сурвалжаас суулгах"-ыг зөвшөөрөх шаардлагатай).

### Зам B — Өөрийн компьютер дээр (Android Studio)

1. [Flutter SDK](https://docs.flutter.dev/get-started/install) болон
   Android Studio суулга.
2. Терминал дээр энэ хавтас руу ор, дараах командыг ажиллуул:

   ```bash
   bash setup.sh
   flutter build apk --release
   ```

   (`setup.sh` нь платформын хавтас үүсгээд, микрофоны зөвшөөрлийг
   тохируулна.)
3. Гарсан .apk энд байрлана:
   `build/app/outputs/flutter-apk/app-release.apk`

## Апп-ыг ашиглах

1. Эхлүүлэхэд **Тохиргоо** дэлгэц нээгдэнэ → 2 API түлхүүрээ оруул.
   Хүсвэл "Өнгө аяс / зан төлөв" талбарт хариулт ямар маягаар гарахыг
   бич (ж: "хошин шогтой, найзархаг ярь").
2. **Хоолой бүртгэх** дэлгэцэнд 15–30 сек тод яриад "Хоолойг
   хуулбарлах" дар.
3. Үндсэн дэлгэцэнд том микрофон товчийг дараад асуу, дахин дараад
   зогсоо — хариулт дуугаар сонсогдоно.

Дээд талын цэснээс хоолойг дахин бүртгэх, тохиргоо солих боломжтой.

---

## Хязгаарлалт / анхаарах зүйл

- **Интернэт шаардана** — энэ нь cloud үйлчилгээ ашигладаг онлайн хувилбар.
- **Хоолой хуулбарлах чанар** нь бичлэгийн цэвэр байдлаас ихээхэн
  хамаарна. Чимээгүй орчинд, чанга тод яривал илүү сайн гарна.
- **Монгол хэлний дуу гаргалт** загвараас хамаарч аялга бага зэрэг
  өөр гарч болзошгүй. Шаардлагатай бол `voice_service.dart` доторх
  `voice_settings` утгуудыг тохируулж туршаарай.
- Зөвхөн зөвшөөрөл авсан хүний хоолойг хуулбарлаарай — бусдын хоолойг
  зөвшөөрөлгүй ашиглах нь ёс зүй/хуулийн асуудал үүсгэж болзошгүй.

## Бүтэц

```
lib/
  main.dart                  # Эхлэх цэг, дэлгэц сонголт
  services/
    settings_service.dart    # API түлхүүр, voice_id хадгалах
    audio_service.dart       # Бичих / тоглуулах
    stt_service.dart         # Яриа -> текст (Whisper)
    chat_service.dart        # Хариулт (GPT)
    voice_service.dart       # Хоолой клон + TTS (ElevenLabs)
  screens/
    settings_screen.dart     # API түлхүүр тохируулах
    enroll_screen.dart       # Хоолой бүртгэх
    home_screen.dart         # Яриа дэлгэц
AndroidManifest.template.xml # Микрофон/интернэт зөвшөөрөл
setup.sh                     # Платформ үүсгэх туслах скрипт
.github/workflows/build-apk.yml  # Автомат .apk build
```
